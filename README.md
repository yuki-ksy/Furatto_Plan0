# Furatto Plan

## サイト概要
日帰り専用の旅行サイト

### サイトテーマ
ふらっと出掛ける場所を探す

### テーマを選んだ理由
急に出かけたい気分になった時に気軽に検索できるサイト


### ターゲットユーザ
休日を充実させたい人

### 主な利用シーン
休日
急に1日空いてしまった時
ふらっと出かけたい時
